El punto de entrada de la aplicación está en el archivo CharacterController.java, ahí se encuentra el endpoint llamado /api/character/{id}.
Para hacer funcionar la aplicación deben abrir la carpeta del proyecto usando IntelliJ IDEA 2021.3.1 (Community Edition), luego 
deben buscar el archivo ChallengeApplication.java y una vez ahí le dan click al ícono verde con forma de triángulo(linea 7) para iniciar
el servicio desde el IDE. Funciona con el Java 15 que viene embebido en el IntelliJ.
